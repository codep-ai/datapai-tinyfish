# Vision

This app must position DataP.ai as the trust + intelligence layer
on top of TinyFish.

TinyFish = web execution.
DataP.ai = trusted data + AI reasoning + dashboards.

We are NOT building a web automation competitor.
We are building a financial intelligence layer powered by TinyFish.

The demo must:
- Show TinyFish scanning real websites.
- Show structured evidence.
- Show explainable scoring.
- Show price context.
- Avoid buy/sell advice.
